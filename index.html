⭐⭐Error detection and correction⭐⭐

Error: noise can introduce error in the binary bits.
It means '0' may change to '1' OR '1' change to '0'

Types of Error

  • Single Bit error
  • Burst Error

    • single Bit error

    In this only 1 bit in the data unit has changed

    Burst error

    In this 2 or more bit in the data unit changes

    Error Dectection method

    1. Parity checking

    In this one extra bit (parity bit ) is addesd to each word before transmitting

  • Even parity
  • Odd parity

    • Even parity

    • No. of 1's in the given data including parity bit should be even
    • At receiving end if no. of 1's is odd : means there is error
    • else there is no error

    Odd parity

    • No. of 1's in the given data including parity bit should be odd
    • At receiving end if no. of 1's is even: means there is error
    • else there is no error

    Limitations

    • Not suitable for detection of multiple error
    • Can not reveal the location of error
    • Cannot correct the error
    • It only detects error
